# 10 - Impostazioni, Backup e Sicurezza Operativa

## Chapter Metadata

- Primary route(s): `/impostazioni`
- Primary user goal: proteggere dati e mantenere continuita operativa
- Related assistant prompts:
  - `> apri impostazioni`
  - `> verifica backup`

## 1) Cosa puoi fare qui

Gestire backup, restore, export dati e impostazioni base account.

## 2) Passi consigliati

1. Apri `/impostazioni`.
2. Crea backup periodici prima di modifiche critiche.
3. Verifica backup disponibili e checksum quando richiesto.
4. Usa restore solo dopo aver confermato il file corretto.
5. Mantieni ordinata la routine di backup (es. settimanale).

## 3) Errori comuni

- Errore: fare restore senza backup recente di sicurezza.
- Perche succede: operazione urgente senza checklist.
- Come risolvere: crea sempre un backup pre-restore.

## 4) Troubleshooting

- Problema: backup presente ma dubbio su integrita.
- Azione: esegui verifica backup prima del ripristino.
- Problema: dopo restore sessione incoerente.
- Azione: effettua nuovo login e ricontrolla i dati principali.
