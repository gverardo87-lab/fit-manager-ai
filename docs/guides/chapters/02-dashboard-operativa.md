# 02 - Dashboard Operativa

## Chapter Metadata

- Primary route(s): `/`
- Primary user goal: capire in pochi secondi cosa fare adesso
- Related assistant prompts:
  - `> mostra cosa fare oggi`
  - `> marco domani alle 18 pt`

## 1) Cosa puoi fare qui

Vedere stato operativo della giornata: KPI, alert, agenda e azioni rapide.

## 2) Passi consigliati

1. Leggi i KPI in alto per un check rapido del carico lavoro.
2. Controlla il pannello alert e apri i dettagli quando necessario.
3. Scorri "Agenda Oggi" per confermare priorita e stati appuntamenti.
4. Usa "Azioni rapide" per creare subito cliente/contratto/scheda/evento.
5. Se sei al primo avvio, usa il blocco "Primi passi".

## 3) Errori comuni

- Errore: usare la dashboard come archivio completo.
- Perche succede: la dashboard e' un cockpit, non una lista dettagliata.
- Come risolvere: usa i link rapidi verso Agenda, Clienti o Contratti.

## 4) Troubleshooting

- Problema: non vedo dati attesi.
- Azione: usa "Riprova" nei blocchi con errore e verifica filtri/stato nelle pagine dominio.
- Nota: i dati economici dettagliati sono in `/cassa`, non nella dashboard.
