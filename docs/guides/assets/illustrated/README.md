# Illustrated Assets - Wave 2

Cartella di riferimento per screenshot annotati della guida illustrata.

## Regole file

- formato consigliato: `png`;
- naming: `<flow-id>-<step-id>-<viewport>.png`;
- viewport:
  - `desktop`
  - `tablet`
  - `mobile`

Esempio:

- `F02-S01-desktop.png`
- `F02-S01-tablet.png`
- `F02-S01-mobile.png`

## Privacy

- non usare dati reali identificabili;
- evitare importi economici sensibili fuori dai flussi cassa;
- oscurare contatti personali quando non necessari allo step.
