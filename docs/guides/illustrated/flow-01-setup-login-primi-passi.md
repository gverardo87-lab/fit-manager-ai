# Flow 01 - Setup, Login e Primi Passi

- Flow ID: `F01`
- Capitolo collegato: `01-inizio-rapido.md`
- Obiettivo: portare l'utente da primo avvio a primo cliente creato.

## Step Frames

| Frame ID | Viewport | Azione | Risultato atteso |
|---|---|---|---|
| F01-S01 | desktop/tablet/mobile | Compila setup account | Account creato |
| F01-S02 | desktop/tablet/mobile | Esegui login | Accesso dashboard |
| F01-S03 | desktop/tablet/mobile | Usa "Primi passi" -> Nuovo cliente | Primo cliente salvato |

## Callout Mapping

### F01-S01

1. Apri schermata setup
2. Compila campi obbligatori
3. Conferma creazione account

### F01-S02

1. Inserisci email account
2. Inserisci password account
3. Premi Accedi

### F01-S03

1. Apri card Primi Passi
2. Seleziona Aggiungi cliente
3. Conferma salvataggio cliente
