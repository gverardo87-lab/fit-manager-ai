# FitManager Guide - Wave 2 (Illustrated Layer)

Questa sezione estende la guida Wave 1 con walkthrough visuali orientati all'azione.
Ogni flow include:

- frame per desktop/tablet/mobile;
- callout numerati (max 4 per frame);
- azione attesa e risultato verificabile.

## Standard applicati

- max 4 callout per frame;
- label callout 3-7 parole;
- set responsive per ogni flow:
  - desktop (>=1024px)
  - tablet (~768px)
  - mobile (~390px)
- esempi privacy-safe (niente dati personali reali).
