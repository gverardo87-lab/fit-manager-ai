# FitManager Guide - Wave 1 (Chapters and Sections)

Guida operativa pensata per accompagnare l'utente in modo accogliente e pratico.
Ogni capitolo e' orientato all'azione e include:

- cosa puoi fare in quella pagina;
- passi chiari da seguire;
- errori comuni;
- soluzioni rapide.

## Come usare questa guida

1. Parti da `01-inizio-rapido.md` se sei al primo accesso.
2. Usa i capitoli per area operativa durante il lavoro quotidiano.
3. Se sei bloccato, apri `12-faq-e-troubleshooting.md`.
