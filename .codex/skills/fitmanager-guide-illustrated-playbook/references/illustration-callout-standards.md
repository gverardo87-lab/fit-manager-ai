# Illustration Callout Standards

## Callout Format

- Marker: numeric badge (`1`, `2`, `3`)
- Label: short phrase (3-7 words)
- Purpose: action verb + object

Example:

- `1` "Apri Nuovo Cliente"
- `2` "Compila campi obbligatori"
- `3` "Conferma salvataggio"

## Density Limits

- max 4 callouts per image;
- if more are needed, split into additional frames;
- avoid placing callouts in corners with tooltips or sticky buttons.

## Responsive Set

For critical pages provide:

- desktop frame (>=1024px);
- tablet frame (~768px);
- mobile frame (~390px).

## Privacy and Safety

- blur or hide personal contact data when possible;
- do not show financial totals in general overview examples;
- avoid real customer names in screenshots.
